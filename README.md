# LMS - Learning Management System

This is a JavaFX-based project with:
- User Registration
- Role-based Dashboards (Admin, Instructor, Student)
- Database integration (PostgreSQL)
- Secure Login

## Technologies
- JavaFX
- PostgreSQL
- JDBC
- Scene Builder
